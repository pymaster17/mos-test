Thanks for participating in this Prosody Similarity (PS-CMOS) listening test. It will take roughly 10-15 minutes. Please use good headphones.

Question: First listen to the Reference singing. Then, ignoring vocal characteristics (timbre, style, vocal techniques), decide which singing's prosody (rhythm, melody) is more consistent and identical with the reference.
  • Each test provides a Reference plus two singing samples, A and B.
  • Compare prosody similarity to the reference and choose one option on the 5-point scale.
  • A +2 = A is much more similar, A +1 = A is slightly more similar, Tie = equally similar, B +1 / B +2 = B is slightly / much more similar.
