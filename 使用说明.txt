1.直接使用：利用docker将mos挂载到服务器上：
使用命令：
	docker run -d -p 80:80 -v D:/你的文件夹存放路径/mos:/var/www/html --name mos-test-server php:7.0-apache
2.目录说明：
audio文件夹：存放你的音频
——文件夹：vocos_mos_test：gt、hifigan、spiking、vocos四个子文件夹，各50个音频片段（觉得数量不够可继续加）
——文件夹：vocos_mos_set:存放set集合，用于呈现到网页上，共50个set，每个里面有gt、hifigan、spiking、vocos四段音频
——文件organize_audio_files.py:编写的快速生成set集合的程序：
使用命令：
	python organize_audio_files.py
	提示输入：
		包含四个音频文件夹的基目录路径
		输出目录路径（将在此创建set文件夹）
——文件generate_config.py:快速编写config配置文件，根据vocos_mos_set文件夹中的set数量快速生成配置文件。
使用命令：
	python generate_config.py
	按照提示输入：
		包含set文件夹的目录路径
		输出配置文件路径
——config文件夹：存放new_config_mushra.js配置文件（可修改）
——index_new.html：网页主页面，可修改里面的文件和测试类别
修改配置类，替换对应的.js配置文件就行：
	<script src="config/mos_test_config.js" type="text/javascript"></script>
修改测试类别，想用哪个类别，取消前面注释就行，：
	<script type="text/javascript">
  	  var testHandle;
  	  window.onload=function() {
   	  testHandle = new MushraTest(TestConfig); // 使用 MUSHRA 测试类
 	 };
	</script>
